# deleteBackup


## 描述
删除备份

## 请求方式
DELETE

## 请求地址
https://mongodb.jdcloud-api.com/v1/regions/{regionId}/backups/{backupId}

|名称|类型|是否必需|默认值|描述|
|---|---|---|---|---|
|**backupId**|String|True| |backup ID|
|**regionId**|String|True| |Region ID|

## 请求参数
无


## 返回参数
|名称|类型|描述|
|---|---|---|
|**requestId**|String| |


## 返回码
|返回码|描述|
|---|---|
|**200**|OK|
