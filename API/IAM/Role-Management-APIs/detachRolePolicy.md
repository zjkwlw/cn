# detachRolePolicy


## 描述
为角色绑定策略

## 请求方式
DELETE

## 请求地址
https://iam.jdcloud-api.com/v1/role/{roleName}/policy/{policyName}

|名称|类型|是否必需|默认值|描述|
|---|---|---|---|---|
|**policyName**|String|True| |策略名称|
|**roleName**|String|True| |角色名称|

## 请求参数
无


## 返回参数
|名称|类型|描述|
|---|---|---|
|**requestId**|String| |



## 返回码
|返回码|描述|
|---|---|
|**200**|OK|
