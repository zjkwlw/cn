
# 产品概述

API 网关（API Gateway），是API托管服务。 提供API的全生命周期管理，包含发布、管理、运行、维护、下线等。用户可通过API网关实现自身系统集成和服务聚合，还能便捷安全的开放其业务功能和数据，并实现与开发者或合作伙伴的连接。
