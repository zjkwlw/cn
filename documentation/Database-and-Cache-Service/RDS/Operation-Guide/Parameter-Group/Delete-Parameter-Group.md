# 删除参数组
随着需求变更，有一些参数组不再使用，您可以手动删除无用的参数组。

## 注意事项
* 参数组未绑定任何实例的情况下才允许删除。

## 操作步骤
1. 登录 [参数组控制台](https://rds-console.jdcloud.com/paramgroup/list)。
2. 在参数组列表页，选择需要拷贝的源参数组，点击 操作 这一列的 删除 按钮。
3. 删除确认弹出框如图
    * 点击 ***确认*** 按钮，完成删除参数组操作。
    * 点击 ***取消*** 按钮，放弃删除参数组操作。
