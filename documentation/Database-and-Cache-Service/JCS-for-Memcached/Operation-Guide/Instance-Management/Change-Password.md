# 修改密码

如果您需要修改旧密码、或者忘记密码，您可以重新设置实例的密码。

1. 登录Memcached控制台

2. 选择目标实例，点击实例名称进入实例详情页面

3. 点击右上角[操作]下拉选择修改密码，弹出窗口输入新密码，确认密码后点击确定
