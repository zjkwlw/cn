# 产品优势

## 简单易用
一键式集群创建
## 高可靠存储引擎	
分布式集群架构、至少3副本架构保障数据可靠性。
## 服务高可用	
HA实时监控系统，单节点故障迅速转移，保障业务不中断 。
## 兼容生态产品	
支持phoenix，支持openTSDB
