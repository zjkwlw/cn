# 查询消息

![查询1](https://github.com/jdcloudcom/cn/blob/edit/image/Internet-Middleware/Message-Queue/查询-01.png)
选取想要查询的topic名称和时间范围，点击搜索，得到查询结果。  
可以点击某条message，查看详情
