# 概览
操作保护是指当您在进行敏感操作的时候，在操作最终执行前要求用户进行身份验证，最大程度的防止对资源的误删和误操作，当前支持对主账号和子账号单独设置，提供三种身份验证方式：MFA验证，短信验证，邮箱验证。
