## 添加解析

1、进入控制台界面后

![img](https://github.com/jdcloudcom/cn/blob/dns-zhangjingfeng/image/dns-img/add-record1.png) 

2、点击此域名或右侧的解析按钮进入域名解析页面

![img](https://github.com/jdcloudcom/cn/blob/dns-zhangjingfeng/image/dns-img/add-record1.png)  

3、点击添加解析后弹出窗口

![img](https://github.com/jdcloudcom/cn/blob/dns-zhangjingfeng/image/dns-img/add-record1.png)  
 
4、根据您的域名填写对应的记录
