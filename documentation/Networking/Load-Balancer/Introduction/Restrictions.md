# 限制说明

您可以快速创建并使用应用负载均衡，但使用中有部分约束条件需要注意。


| 资源	| 限制	| 例外申请方式 |
| :- | :- | :- |
|单地域下应用负载均衡实例	|5个	|工单|
|一个应用负载均衡下的监听器	|20个	|无|
|一个应用负载均衡下的后端服务	|20个	|无|
|一个应用负载均衡下的虚拟服务器组	|20个|	无|
|一个虚拟服务器组内的实例	|100个|	无|
